Project Site Plan

Site Name

The Ogutu Empire

This name represents a family website designed to document and share family history, achievements, and special events.

Site Purpose

The site provides a digital space for the Ogutu family to stay connected. It will include family trees, photo galleries, important announcements, and family achievements.

Scenarios

  • How can I view the family tree and learn about my ancestors?
  • Where can I find the latest family event details?

Color Schema

  • #2c3e50 (Dark Blue) - Used for headings and navigation.
  • #ecf0f1 (Light Gray) - Background color.
  • #333 (Dark Gray) - Body text.

Typography

  • Arial - Used for body text.
  • Georgia - Used for headings.
